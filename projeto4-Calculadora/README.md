# 🔢 Calculadora de Operações Básicas

Este projeto consiste em uma calculadora funcional desenvolvida para consolidar o aprendizado de funções e lógica aritmética em ambiente web.

## 🎯 Objetivo
Desenvolver uma interface simples e intuitiva que realize cálculos matemáticos em tempo real, focando na organização do código e na experiência do usuário.

## 🧠 Lógica e Funcionalidade
Neste projeto, foquei nos seguintes conceitos de programação:

1. **Funções Matemáticas:** Criação de funções específicas para soma, subtração, multiplicação e divisão.
2. **Escopo de Variáveis:** Gerenciamento dos valores de entrada para garantir que os cálculos sejam precisos.
3. **Eventos de Clique:** Uso de `addEventListener` para capturar as ações do usuário nos botões da calculadora.
4. **Tratamento de Dados:** Conversão de strings (texto) para números, garantindo que as operações aritméticas ocorram corretamente no JavaScript.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura dos botões e do visor da calculadora.
* **CSS3:** Design moderno, utilizando Flexbox para o alinhamento dos componentes.
* **JavaScript:** Toda a parte lógica e o processamento das operações.
