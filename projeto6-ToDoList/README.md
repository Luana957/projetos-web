# 📝 Projeto: Lista de Tarefas (To-Do List)

Este projeto demonstra a manipulação dinâmica de elementos HTML através do JavaScript.

## 🎯 Objetivo Didático
Ensinar como o navegador pode "criar" novos elementos na página em tempo real (Manipulação de DOM) a partir da interação do usuário.

## 🧠 Conceitos Aplicados:
1. **createElement**: Criação de novos elementos `<li>` via script.
2. **appendChild**: Inserção de novos itens dentro de uma lista existente.
3. **Remoção de Elementos**: Lógica para deletar uma tarefa específica ao clicar no botão.
4. **Validação Simples**: Verificação para não adicionar tarefas vazias.
