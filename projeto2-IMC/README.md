# 🧮 Calculadora de IMC - Projeto Didático

Este projeto foi desenvolvido para aplicar conceitos fundamentais de programação web, focando em interatividade e lógica de processamento de dados.

## 🎯 Objetivo
Criar uma ferramenta funcional que auxilie o usuário a calcular seu Índice de Massa Corporal de forma rápida, demonstrando o uso prático de JavaScript no navegador.

## 🧠 Lógica e Funcionalidade
Para a construção deste projeto, apliquei os seguintes conceitos técnicos:

1. **Captura de Dados (DOM):** Utilização de métodos do JavaScript para ler os valores de peso e altura inseridos nos campos de input.
2. **Cálculo Matemático:** Implementação da fórmula padrão: `IMC = peso / (altura * altura)`.
3. **Estruturas Condicionais:** Uso de blocos `if/else` para validar o resultado e retornar a classificação correta (Abaixo do peso, Peso normal, Sobrepeso, etc).
4. **Feedback Visual:** Alteração dinâmica do HTML para exibir o resultado final para o usuário sem a necessidade de recarregar a página.
